# 🎮 Pixel Character — Sprite Movement

A 2D pixel-art character controller: walk a sprite around a tiled map with directional animations (up / down / left / right).

## Features
- Sprite-sheet animation per direction
- Tiled map background and keyboard movement
- Foundation for a top-down 2D game

## Run locally
Clone the repo and open `index.html` in any browser.

## Tech
`HTML5` · `CSS3` · `JavaScript` · Sprite animation
